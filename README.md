# 노동법률사무소 이해 홈페이지

24시간 AI 노무 상담을 제공하는 정적 웹사이트입니다. 부당해고, 임금체불, 퇴직금, 산재, 직장 내 괴롭힘 등 노동 관련 궁금증을 언제든지 물어볼 수 있도록 만들어졌습니다. Google Gemini API를 사용하며, 사용자가 본인의 API 키를 입력하는 방식이라 별도의 서버나 회원가입이 필요 없습니다.

> 본 사이트는 일반 시민과 근로자가 손쉽게 노무 정보를 얻을 수 있도록 안내하는 참고용 도구입니다.

---

## 주요 기능

- **24시간 AI 노무 상담**: 부당해고, 임금체불, 퇴직금 계산, 산업재해, 직장 내 괴롭힘 등 폭넓은 주제 응답
- **API 키 안전 관리**: 사용자 본인의 Google Gemini API 키를 입력 → **브라우저 localStorage에만 저장** (외부 서버 전송 없음)
- **반응형 UI**: 데스크톱, 태블릿, 모바일에서 모두 정상 동작
- **빌드 불필요**: HTML/CSS/JS 정적 파일로만 구성되어 어떤 정적 호스팅에도 그대로 올릴 수 있습니다

## 실행 방법

### 방법 A — `index.html` 더블클릭 (가장 쉬움)

Finder에서 `index.html` 파일을 더블클릭하면 기본 브라우저로 열립니다. 가장 빠르지만, 일부 브라우저는 `file://` 환경에서 fetch 요청을 차단할 수 있습니다. 만약 API 호출이 안 되면 아래 **방법 B**로 넘어가세요.

### 방법 B — 로컬 서버 띄우기 (권장)

터미널을 열고 프로젝트 폴더로 이동합니다.

```bash
cd ~/projects/labor-law-office
```

macOS에 기본 설치된 Python으로 간단한 정적 서버를 띄웁니다.

```bash
python3 -m http.server 8080
```

브라우저에서 다음 주소를 엽니다.

```
http://localhost:8080
```

종료할 때는 터미널에서 `Ctrl + C`를 누르세요.

## Gemini API 키 발급 방법

1. https://aistudio.google.com/app/apikey 접속
2. Google 계정으로 로그인
3. 우측 상단의 **"Create API key"** 버튼 클릭
4. 발급된 키(예: `AIzaSy...`로 시작하는 문자열) 복사
5. 사이트의 API 키 입력란에 붙여넣고 저장

> ⚠️ **API 키는 비밀번호와 동일하게 다뤄주세요.**
> - 화면 캡처를 SNS·이메일·메신저로 공유하지 마세요.
> - 무료 한도를 초과하면 과금될 수 있습니다. Google AI Studio의 결제 설정과 사용량을 주기적으로 확인하세요.
> - 키가 노출된 것 같으면 즉시 위 주소에서 키를 삭제하고 새로 발급받으세요.

## 배포 방법 (선택)

배포 없이도 로컬에서 사용할 수 있지만, 다른 사람에게 공유하고 싶다면 아래 무료 옵션 중 하나를 선택하세요.

### 옵션 1 — GitHub Pages (무료, 추천)

1. GitHub에 새 저장소 생성 후 프로젝트 폴더의 모든 파일을 push
2. 저장소 **Settings → Pages**로 이동
3. Source를 `main` 브랜치 / `/ (root)`로 지정 후 저장
4. 1~2분 후 `https://USERNAME.github.io/REPO` 주소에서 접속 가능

장점: 완전 무료, HTTPS 기본 제공. 다만 공개 저장소면 누구나 코드를 볼 수 있으니 **API 키는 절대 코드에 하드코딩하지 마세요** (현재 구조는 사용자가 본인 키를 입력하는 방식이라 안전).

### 옵션 2 — Netlify Drop (드래그 앤 드롭, 가장 빠름)

1. https://app.netlify.com/drop 접속
2. 프로젝트 폴더(`labor-law-office`)를 통째로 드래그
3. 즉시 임시 URL이 발급되어 공유 가능

배포 후 커스텀 도메인 연결도 무료로 지원됩니다.

### 옵션 3 — Vercel

1. `npm i -g vercel` 후 프로젝트 폴더에서 `vercel` 실행, 또는
2. Vercel 대시보드에서 GitHub 저장소를 연결하면 push할 때마다 자동 배포

GitHub Pages와 비슷하지만 빌드/프리뷰 기능이 더 풍부합니다.

## 자주 묻는 질문 (FAQ)

**Q1. API 키를 입력했는데 답변이 안 와요.**
- 키 형식이 `AIzaSy`로 시작하는지 확인하세요.
- Google AI Studio에서 결제 설정이 활성화되어 있는지 확인하세요 (무료 한도 초과 시 차단됨).
- 브라우저 개발자 도구 콘솔(F12)에서 에러 메시지를 확인해보세요.

**Q2. 제 API 키가 외부로 새지 않나요?**
- 모든 통신은 **사용자 브라우저 → Google Gemini 서버**로 직접 이루어집니다.
- 본 사이트는 자체 백엔드 서버가 없으며, 입력한 API 키는 사용자 브라우저의 localStorage에만 저장됩니다.
- 다른 기기·브라우저에서는 다시 입력해야 합니다.

**Q3. 답변이 법적 효력이 있나요?**
- 아니요. AI가 생성한 일반적인 정보 안내일 뿐입니다.
- 정확하고 구속력 있는 자문은 반드시 **노무사 또는 변호사와 직접 상담**하세요.

**Q4. 다른 Gemini 모델로 바꿀 수 있나요?**
- `js/gemini-api.js` 상단의 모델명 상수만 교체하면 됩니다 (현재 `gemini-3-flash-preview`).
- 단, 모델별로 응답 속도·비용·정확도가 다르니 변경 후 충분히 테스트해보세요.

**Q5. 비용은 얼마나 나오나요?**
- Google Gemini API의 **무료 한도** 안에서는 무료입니다.
- 한도 초과 시 사용자 본인 키로 과금되므로, AI Studio의 사용량 페이지를 주기적으로 확인하세요.

## 커스터마이징 안내

- **사무소 이름·연락처 변경**: `index.html`의 헤더(`<header>`) 및 푸터(`<footer>`) 영역
- **색상·폰트 변경**: `styles.css` 상단의 CSS 변수(`:root { --primary: ... }`) 부분
- **AI 답변 톤·역할 변경**: `js/gemini-api.js`의 `SYSTEM_PROMPT` 상수 (예: "친절한 노무사처럼 답변" 등으로 수정)
